# RAMAN-LINK // Automated Spectral Assignment

## Overview
**RAMAN-LINK** is an analytical tool designed for the interpretation and assignment of Raman spectra. Optimized for **Organic APIs and Pharmaceutical Materials**, the platform correlates raw laboratory data with molecular structure to streamline the identification of structural markers and polymorphic forms.

## Core Features
1.  **Instrument Agnostic**: Ingest raw data from Bruker, Renishaw, Horiba, and other major spectrometers without manual formatting.
2.  **Structural Correlation**: Automated mapping of spectral peaks to molecular functional groups.
3.  **Metadata Extraction**: Instant retrieval of polymorphic reference markers from research indices.
4.  **Deterministic Signal Chain**: Standardized math for despiking, baseline correction, and smoothing.

## Methodology
The platform utilizes a fragment-based approach to spectral assignment. By analyzing the molecular connectivity (SMILES), the engine predicts characteristic vibrational frequencies grounded in established spectroscopic standards (including the Socrates characteristic group frequencies).

## How to Run
This is a static web interface.
1. Clone the repository.
2. Open `index.html` in any modern browser.
